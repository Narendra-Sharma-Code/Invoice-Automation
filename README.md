Installation and run steps

Step 1:
Check python version, must be 3.10.2
`
python --version
`

Step 2:
Create & Activate Virtual Environment
`
pip install venv
python -m venv invauto_env
.\invauto_env\scripts\activate
`

Step 3:
Install Requirements
`
pip install requirements.txt
`

Step 4:
Run app
`
python run.py
`
